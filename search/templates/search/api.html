https://ahr9n-quran-api.herokuapp.com/api/all-verses

returns all verses.

https://ahr9n-quran-api.herokuapp.com/api/search-word/{words}

takes a string to return all verses, containing this string.

https://ahr9n-quran-api.herokuapp.com/api/get-surah/{surah_id}

takes Surah number, to return all verses in this Surah. Notice that Surah numbers is between 1 and 114.

https://ahr9n-quran-api.herokuapp.com/api/get-verse/{surah_id}/{verse_id}

returns a specific verse by taking surah number and verse number.
